Quick Start Guide

  1. Find Dealers: Enter a dealer ID to view citation history and available citations
  2. Get Suggestions: The system will show every citation they haven't used in the last 6 months
  3. Add Citation: Once you've built a citation, record it to keep the database updated
  4. Check Dashboard: View overall statistics and identify dealers needing citations

Key Features

  • Centralized dealer and citation database
  • 6-month recency rule to prevent citation repetition
  • Intelligent suggestion algorithm
  • Complete audit trail with timestamps
  • Quick lookup by dealer ID
  • Monthly citation tracking
